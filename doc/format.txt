BOX Archives
============

Format v1
---------

	Note, that this format version has limitations, as it is still in development.
	For a manually-created example, see format_v1.box

	Binary file:

	Offset		Length			Value		Description

	0			2				0xDE, 0xCA, 0xDE File Identifier
	3			1				uint8_t		File version  (0x01)
	4			1				0x00		padding
	5			2				uint16_t	header_length
	9 [			spec. at off.	ASCII text  Here goes the header XML file* .
	] 10			2				0x00, 0xFF	End-of-header marker.
	12							Binary data	The file data referred to by offsets stored in the deader XML file.
	13			2				0x00, 0xFF	End of file marker.

	XML Header:
		Allowed tags:
			<header>

			<dir name="" owner="">
				name  = The directory name
				owner = The directory owner

			<file name="" owner="" size="" start="">
				name  = The file name
				owner = The file's owner
				size  = The file's size in bytes.
				start = The offset of the file data
